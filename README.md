
# 🚀 Быстрый старт: Запуск бота на Ubuntu (для новичков)

Если вы **никогда раньше не работали с Ubuntu**, этот гайд написан специально для вас. Всё максимально просто и по шагам.

---

# Как подключиться к серверу

### Вариант А: Через **PuTTY** (рекомендуется для Windows)

1. Скачайте и установите **PuTTY** с официального сайта(ссылка внизу):  
   [https://www.putty.org/](https://www.putty.org/) 

2. Запустите программу.

3. В главном окне заполните:
   - **Host Name (or IP address)** — IP-адрес вашего сервера (например: `123.456.789.012`)
   - **Port** — `22`
   - **Connection type** — **SSH**

4. Нажмите **Open**.

5. В чёрном окне введите:
   - **login as:** — ваш логин (обычно `root`)
   - **password** — пароль от сервера (при вводе он не отображается — это нормально)

Готово! Вы внутри сервера.

---

### Вариант Б: Через терминал (Linux / macOS)

Откройте терминал и выполните:

```bash
ssh ваш_логин@IP_АДРЕС_СЕРВЕРА
```

Пример:
```bash
ssh root@123.456.789.012
```

Введите пароль, когда попросит.

# Установка

После подключения по SSH выполняйте команды по очереди:
- Обновляем пакеты и устанавливаем нужные программы
```bash

sudo apt update
sudo apt install python3 python3-venv python3-pip git screen -y
```
- Создаём папку для бота
```bash
mkdir ~/bot && cd ~/bot
```
- Скачиваем бота
```bash
git clone https://github.com/adoubt/41_michaelbot
```
- Создаём виртуальное окружение
```bash
python3 -m venv venv
source venv/bin/activate
```
- Устанавливаем зависимости
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### Создайте файл настроек `.env`

Выполните команду:

```bash
cat > .env << EOF
PASSWORD=любой_пароль_для_админа
BOT_TOKEN=ваш_токен_бота_от_BotFather
EOF
```

(Замените `любой_пароль_для_админа` и `ваш_токен_бота_от_BotFather` на свои реальные данные)

# Запуск бота
- Устанавливаем пакет для фонового запуска
```bash
sudo apt update
sudo apt install screen -y
```

- Запускаем сессию screen
```bash
screen -S lisabot
```
- Запускаем бота
```bash
python3 main.py
```

Чтобы **отсоединиться** от сессии (бот продолжит работать):
Нажмите `Ctrl + A`, потом `D`

# Обновление бота
- Открываем папку с ботом, если еще не тут.
```bash
cd ~/bot
```
- Возвращаемся в сессию и останавливаем бота
```bash
screen -r lisabot
```
- Нажмите Ctrl + C

- Скачиваем обновление
```bash
git pull
```

- Обновляем зависимости (на всякий случай)
```bash
source venv/bin/activate
pip install -r requirements.txt
```

- Запускаем заново
```bash
screen -S lisabot
python3 main.py
```
# Ctrl + A, потом D — отсоединиться
